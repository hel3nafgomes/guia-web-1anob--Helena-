O que é HTML, CSS e JavaScript?

HTML

HTML (HyperText Markup Language) é a linguagem de marcação padrão para criar páginas web.

<h1>Título</h1>

Quando usar: Para estruturar o conteúdo da sua página.

CSS

CSS (Cascading Style Sheets) é usado para descrever a apresentação de um documento escrito em HTML.

body { background-color: #F3F4F6; }

Quando usar: Para estilizar e formatar o layout da sua página.

JavaScript

JavaScript é uma linguagem de programação que permite implementar funcionalidades complexas em páginas web.

console.log('Olá, mundo!');

Quando usar: Para adicionar interatividade e dinamismo à sua página.
