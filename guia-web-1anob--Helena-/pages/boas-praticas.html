Boas Práticas em Desenvolvimento Web

Semântica & Acessibilidade

  • Utilizar landmarks para estruturação do HTML.
  • Incluir atributos alt em imagens.
  • Conectar labels a inputs usando for/id.
  • Garantir contraste adequado entre texto e fundo.
<img src="imagem.jpg" alt="Descrição da imagem">

Responsividade

  • Adotar mobile-first na abordagem de design.
  • Utilizar Flexbox ou Grid para layouts.
  • Preferir unidades relativas como % e em.
display: flex;

Organização de CSS

  • Utilizar BEM ou outra convenção de nomenclatura.
  • Adicionar comentários explicativos no código.
  • Definir variáveis CSS para cores e fontes.
:root { --cor-primaria: #2563EB; }

Performance Leve

  • Otimizar imagens para web.
  • Implementar lazy loading para imagens.
  • Evitar JavaScript desnecessário.
<img src="imagem.jpg" loading="lazy">

Versionamento (Git)

  • Escrever mensagens de commit claras e descritivas.
  • Utilizar branches para novas funcionalidades.
  • Fazer commits frequentes para manter o histórico.
git commit -m "feat: adicionar nova funcionalidade"

Checklist Pessoal

  • Revisar semântica e acessibilidade
  • Testar responsividade em dispositivos
  • Organizar CSS e adicionar comentários
  • Otimizar imagens do projeto
  • Fazer commit das alterações
